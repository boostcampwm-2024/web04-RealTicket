# RealTicket

## 프로젝트 개요

- 인터파크 티켓 예매와 같이 특정 시간에 대규모 요청이 들어오는 서비스를 구현한다.
- 대규모 트래픽, 대규모 데이터를 최적화하며 성능 개선 과정을 기록한다.
- 사용자가 예매 연습을 할 수 있도록 해서 사용자의 반응속도를 연습할 수 있도록 한다.
